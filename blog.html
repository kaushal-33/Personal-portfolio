Sticky Header
STICKY HEADER USING WITH JQUERY

A sticky header is a typical element of a lot of contemporary websites. This feature enables the header to remain at the top of the page even as you scroll down, giving you continuous access to the navigation menu and other significant components.

Click here for explanation
  • HTML: The 'header' element will be the sticky header. The (div id="content") is for the main content of the page.
  • CSS: 'header' starts with a relative position.sticky class changes the 'header's position to 'fixed', making it stick to the top when scrolled.
  • JQUERY: 'sticky' is the initial vertical position of the header. On scroll, if the window's scroll position is greater than the header's top position, the (.sticky) class is added to the header. If the scroll position is less, the (.sticky) class is removed.
Pre-loader
HOW TO ADD PRELOADER USING JQUERY

A preloader is a visual element that shows up while your page or specific content loads. Examples of these elements include spinners and progress bars. A loading animation or image is displayed until the page is fully loaded, at which point it is hidden. This is how a preloader using jQuery is implemented.

Click here for explanation
  • HTML: '#preloader' is the preloader element that will be visible during loading. '#content' is the main content of the page that will be shown after loading.
  • CSS: '#preloader' is styled to cover the entire viewport and center to the body. '#content' is hidden initially to avoid displaying the content before the preloader disappears.
  • JQUERY: $(window).on('load', function() {...}); ensures the code runs after all resources (images, scripts, etc.) have fully loaded. $('#preloader').fadeOut('slow', function() {...}); fades out the preloader. $('#content').fadeIn('slow'); fades in the content after the preloader has been hidden.
